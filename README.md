# Terraform_vpc_module
Terraform  VPC module with multiple Privite and Public Subnets in different AZ

# Objects Created 
- VPC 
- Public subnet
- Private subnet
- Internet gateway
- Elastic Ip
- Nat gateway 
- Route tabel/Public route
- Privite route 

# Execution
- terraform init
- terrafrom plan
- terraform apply
- terraform destroy
